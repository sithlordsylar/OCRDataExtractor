File Format Reminder

Only .jpg, .jpeg, .png formats are accepted. Please ensure your photos have the correct format and extension.

Quick Tips

  1. Toggle session mode if you wish to process multiple images at once.
  2. Select your image files (accepted formats: .jpg, .jpeg, .png) using the file chooser.
  3. Click "Scan Images" to run OCR on your images.
  4. Review the scanned lines which are color-coded for easy navigation.
  5. Label each line appropriately (Name, Contact Number, Email) using the dropdown menus.
  6. If in session mode, click "Complete Session" after processing all images to enable data export.
  7. Export your data using CSV or XML, or clear all data if needed.
